_I hope we will always find ourselves beneath the same quiet sky,
hands warmed by the same small fire,
roasting marshmallows as if time itself were kind to us.
And if life, in all its restless cruelty,
should pull us apart before we can taste that sweetness again,
then let the next life be gentler—
let it return us to one another,
to the soft crackle of the flames,
where love waits patiently in the glow._
